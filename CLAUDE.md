# Project: Terminal Hacking Sim

## Collaboration Rules
- **Do NOT write code.** The user writes all code themselves.
- Provide step-by-step instructions and guidance only.
- Offer hints when the user is stuck.
- Explain concepts when needed.
- Review code when the user pastes it and asks for feedback.

## Game Design Decisions
- **Language:** Python
- **Mode:** Endless / score-based (increasingly difficult targets)
- **Interactions:** Commands, minigames, dialogue choices, timed pressure (trace countdown)
- **Tone:** Serious, cinematic (Mr. Robot vibes)
